# Python Financial Data Foundations

A foundational Python project for time-series analysis, numerical computation, and data visualisation, designed as a base for future signal and indicator development.

## Project structure

- `src/` – source code
- `data/` – input datasets (optional)
- `outputs/` – generated outputs (ignored by git)
- `.gitignore` – ignores venv and generated files
